## `whoami`

### Building practical tooling for recon, OSINT and detection-focused security work

I build cybersecurity projects around operator workflow, automation and research. The public work on this account is centered on offensive security utilities, OSINT applications and detection engineering experiments intended for authorised testing and defensive validation.

- `SpectreIntel`: OSINT for cybersecurity professionals
- `DNS-Tunneling-Detection`: Suricata rules, traffic generation and evaluation for DNS tunneling detection
- `Port-Scanner`: Python-based Nmap automation for authorised reconnaissance
- `Agent-Office`: TypeScript workspace for agent-driven workflows
- `Stack`: Python, TypeScript, React, Tauri, Rust, Linux and Bash
- `Current direction`: desktop-first security tooling, clean operator UX and workflow automation
