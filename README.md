I build cybersecurity projects around offensive security, detection engineering and repeatable analyst workflow. Recent work includes threat modelling for authentication systems, detection and monitoring requirements, analyst runbooks, SIEM triage, identity-focused purple team validation and DNS tunneling detection research. I prefer practical projects that turn research into usable tooling and clear operational processes.
